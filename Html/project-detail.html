Moti Motion

Aplikasi musik full stack yang memungkinkan pengguna untuk mencari, mendengarkan, dan mengelola daftar lagu mereka. Dengan menggunakan API musik, aplikasi ini memberikan pengalaman pengguna yang interaktif dan menyenangkan. Pengguna dapat membuat akun, menyimpan lagu favorit, dan berbagi daftar putar dengan teman-teman mereka. Aplikasi ini dirancang dengan prinsip MVC untuk memisahkan logika bisnis, tampilan, dan data, sehingga mudah untuk dikembangkan dan dipelihara.
